# Manual Smoke Test Checklist — eMachination CAD Patent Prompt-Rig

1. Start `launchers\Start eMachination CAD Patent Prompt-Rig.bat`.
2. Press `ScrollLock` and confirm the board opens.
3. Press `NumLock` until each of the six profiles appears.
4. Press `Pause / Break` and confirm board cycling.
5. Trigger `mechanical_drawing_cleanup.primary.01` and confirm the copied prompt is **Clean Drawing**.
6. Edit one prompt file, focus the board, press `F5`, and confirm the changed prompt reloads.
7. Open Prompt Forge and confirm it still opens.
8. Stop with `launchers\Stop eMachination CAD Patent Prompt-Rig.bat`.
