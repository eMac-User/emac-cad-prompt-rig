# Windows Install — eMachination CAD Patent Prompt-Rig

Windows is the primary/current best-supported runtime path for eMachination CAD Patent Prompt-Rig.

## Requirements

- Windows 10 or Windows 11.
- AutoHotkey v2.
- A stable extracted folder location.

## Start

Run:

```text
launchers\Start eMachination CAD Patent Prompt-Rig.bat
```

Then press `ScrollLock`.

## Stop

Run:

```text
launchers\Stop eMachination CAD Patent Prompt-Rig.bat
```

## Edit Prompts

Edit Markdown files under `prompts/`, then press `F5` while the board is focused.
