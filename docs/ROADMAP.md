# Roadmap — eMachination CAD Patent Prompt-Rig

## v0.1 Design Lock

- Same Prompt-Rig runtime shape.
- New CAD / patent prompt library.
- No real image assets required yet.
- Six profiles.
- Three boards per profile.
- Eight slots per board.
- Windows release remains primary.
- Linux source remains included but pragmatic.
- Prompt Forge stays available.
- The board preserves design intent, not ugly geometry.

## Later Candidates

- Add design-patent-specific drawing prompts.
- Add profile-specific board display names in the runtime.
- Add example CAD/patent images after final branding is available.
- Add richer patent drawing review checklists tied to formal source citations.
- Add optional CAD-drafter handoff examples for Fusion, Inventor, and SolidWorks.
