# Design Summary — eMachination CAD Patent Prompt-Rig

This product keeps the small Prompt-Rig runtime model and swaps the active prompt library to CAD / mechanical / patent drawing work.

## Preserved Runtime Shape

- Windows AutoHotkey v2 runtime remains in `src/eMachination_Coding_Prompt_Rig.ahk`.
- Prompt routing remains in `prompts/index.ini`.
- Prompt files remain plain Markdown.
- Prompt Forge and Prompt Coach remain available.
- Brand image paths tolerate `.txt` stubs.
- Linux runtime/source files remain included.

## Active Model

- Profiles: 6
- Boards per profile: 3
- Prompt slots per board: 8
- Active prompts: 144

## Legal/Patent Guardrail

The product can prepare drawing instructions and risk checklists, but it must not claim legal compliance, guarantee patent-office acceptance, or provide legal advice.
