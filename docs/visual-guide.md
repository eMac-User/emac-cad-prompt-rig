# Visual Guide — eMachination CAD Patent Prompt-Rig

This no-images package preserves image locations with `.txt` stubs. Replace the stubs later with real CAD / patent branding screenshots or artwork without changing documentation paths.

## Board Flow

```text
ScrollLock → board opens
NumLock → cycles profiles
Pause / Break → cycles boards
F5 → reloads prompt edits
```

## Recommended First Visual Asset

Create a clean black-and-white mechanical drawing example for the **Mechanical Drawing Cleanup / Clean Drawing** prompt. That image should show how a crooked rough concept becomes a square, readable, CAD-traceable mechanical reference.
