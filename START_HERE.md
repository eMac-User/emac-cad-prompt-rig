## Windows

Run the AutoHotkey board from the Windows launcher/source:

```text
launchers\Start eMachination CAD Patent Prompt-Rig.bat
```

Press `ScrollLock` to open the board.

Use `NumLock` to cycle the six CAD / patent profiles and `Pause / Break` to cycle the three boards.

## What To Try First

1. Open **Mechanical Drawing Cleanup**.
2. Use **Clean Drawing** when you have a messy sketch, CAD screenshot, or AI-generated mechanical concept.
3. Move to **CAD Application Assist** when the part needs Fusion, Inventor, or SolidWorks instructions.
4. Move to **Patent Drawing Standards** when the concept needs patent-style figures, reference numerals, section views, or hatching review.

## Linux

Extract or clone the repo into the folder where you want to keep it.

Open a terminal in that folder and run:

```bash
bash install_linux.sh
```

Do not move or delete the folder after installation because the systemd user hotkey service points back to that folder.
